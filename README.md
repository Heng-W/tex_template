# Tex paper templates


论文Tex模板，包含图片、表格、公式、自定义环境等示例，
使用XeLaTex编译，编译两次（第一次生成标签，第二次引用标签并生成位置链接），
适用于课程设计报告、毕业论文、双栏论文等。
